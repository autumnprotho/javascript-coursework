Conceptual Understanding

Before completing this assignment, I had heard of JavaScript data types but had never spent much time thinking about how they actually work. I learned that JavaScript has several primitive data types, including Number, String, Boolean, Undefined, and Null. Each one serves a different purpose. Numbers are used for calculations, strings store text, booleans represent true or false values, undefined means a variable has been declared but not given a value yet, and null is used when you intentionally want a variable to have no value.

The part that surprised me the most was type conversion. I didn't expect JavaScript to convert certain values the way it does. For example, when I converted null to a number, the result was 0. I also learned that JavaScript will sometimes automatically convert values between types, which can lead to unexpected results if you are not paying attention.
